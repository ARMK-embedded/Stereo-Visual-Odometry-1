## Dependnicies:
```
  OpenCV2.4
  PCL
  g2o
```
## Building Project in terminal:
```
  mkdir build
  cd build
  cmake ..
  make
```

##  Running project
```
cd <project_dir>
./vo PATH_TO_LEFT_IMAGE_SET_DIRECTORY PATH_TO_RIGHT_IMAGE_SET_DIRECTORY PATH_TO_YAML_FILE
```
